* BookmarkCreator
-+-+-+-+-+-+-+-+-+-+-+-+--+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-

   バージョン:
               BookmarkCreator 1.0.0
      著作者 :
               Yor-Jihons

         用途:
               定義ファイルから(ブックマークとしての)HTMLファイルを生成すること。

  できること :
               * -gオプションを使ってHTMLのテンプレートファイルを生成すること
               * 定義ファイルからブックマークとしてのHTMLファイルを生成すること

インストール :
               このzipファイルを解凍すれば完了

    動作環境 :
               @ Windows 7以降

    開発環境 :
               @ Windows 11
               @ C#.NET (.NET5)
